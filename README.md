# 这是一个通过flask实现外卖网站后端的项目
## 此项目使用flask设计一个类似美团的外卖网站的后端

## 接口文档
### [美团接口文档](https://documenter.getpostman.com/view/18742402/UVkmPwcb)

## 项目主要功能介绍
### 1. 用户
#### * 注册、登录、注销
#### * 修改各类资料，头像上传
#### * 修改密码（提高->需要邮箱验证）
#### * 用户间可以相互关注（提高->好友系统）
#### * 点餐
### 2. 商家
#### * 上传物品信息，包括物品图片（20级还需可上传视频）
#### * 商家标签分类
#### * 评论（提高->多级评论）
#### * 点赞、收藏、转发
#### * 商家、物品搜索（20必做）
### 3. 骑手
#### * 接单（为简化系统，可直接定时送达）
#### * 收益
### 4. 后台
#### * 搜索各类信息
#### * 审核删除各类信息
